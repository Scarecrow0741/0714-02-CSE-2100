# Tetris Game - Download & Installation Guide

This guide will walk you through downloading and installing the Tetris game on your computer.

## Step 1: Access the Releases Section

1. Navigate to the **Releases** section of this repository
2. Look for the latest release version (e.g., **Tetris 1.0.0**)
3. Click on the release to view its details

## Step 2: Download the Setup File

1. In the release details, locate the **Assets** section at the bottom
2. Find and click on the **mysetup.exe** file to download it
3. Wait for the download to complete (the file will appear in your Downloads folder by default if you didn't specify a different location)

## Step 3: Run the Setup Installer

1. Navigate to where you downloaded **mysetup.exe** (usually your Downloads folder)
2. Double-click on **mysetup.exe** to launch the setup installer
3. Follow the on-screen prompts:
   - Choose your installation directory (or use the default location)
   - Click **Install** to begin the installation
4. Wait for the installation to complete

## Step 4: Install the Game

1. After the initial setup completes, a game installer/setup dialog will open
2. This second setup wizard will help you install the actual Tetris game
3. Follow the prompts to:
   - Choose your preferences (if any options are available)
   - Select the installation location
   - Click **Install** to proceed
4. Wait for the game installation to complete

## Step 5: Play Tetris!

1. Once the game installation finishes, you can launch Tetris in one of these ways:
   - **From the Installation Folder:** Navigate to the game's installation directory and double-click the executable

2. Enjoy playing Tetris!

## Troubleshooting

### Setup file won't run
- Ensure you have Windows 7 or later
- Right-click on **mysetup.exe** and select **Run as Administrator**
- Disable antivirus temporarily (some may block setup files)

### Need to uninstall?
- Go to **Control Panel** → **Programs** → **Programs and Features**
- Find "Tetris" in the list
- Click **Uninstall** and follow the prompts

## System Requirements

- **OS:** Windows 7 or later
- **Processor:** 1 GHz or faster
- **RAM:** 512 MB minimum
- **Storage:** 100 MB free disk space
- **Graphics:** Support for SDL2-compatible graphics

## Support

3. Create an issue in the repository with details about your problem
